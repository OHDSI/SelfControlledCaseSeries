# Load the cohort method data from a file

Loads an object of type
[SccsIntervalData](https://ohdsi.github.io/SelfControlledCaseSeries/reference/SccsIntervalData-class.md)
from a file in the file system.

## Usage

``` r
loadSccsIntervalData(file)
```

## Arguments

- file:

  The name of the file containing the data.

## Value

An object of class
[SccsIntervalData](https://ohdsi.github.io/SelfControlledCaseSeries/reference/SccsIntervalData-class.md).
